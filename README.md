# 🩺 메디고(MEDIGO)

**발달장애인 및 고령층 1인 가구**를 위한 병원·약국 **동행 서비스 플랫폼**  
혼자서 병원에 가기 어려운 사람들을 돕기 위해,  
**검증된 동행자(매니저)** 를 매칭하여 **안전하고 신뢰성 있는 의료 접근**을 지원합니다.

---

## 💡 프로젝트 개요

기존의 의료 시스템은 ‘병원 방문’을 스스로 수행할 수 있는 사람을 전제로 설계되어 있습니다.  
하지만 발달장애인, 신체장애인, 고령층 1인가구의 경우  
**접수, 대기, 진료, 수납, 약 수령** 과정에서 큰 어려움을 겪습니다.

**메디고(MEDIGO)** 는 보호자의 부재나 일정상의 제약으로 인해  
직접 동행이 어려운 상황에서도 **대체 보호자(동행자)** 를 안전하게 매칭하여  
환자의 의료 접근성을 높이는 **사회적 돌봄 기반 의료 보조 플랫폼**입니다.

---

## 🚨 서비스 필요성

| 기존 문제 | 메디고의 해결 방식 |
|------------|--------------------|
| 보호자 부재로 인한 병원 방문 어려움 | 인증된 동행자(매니저) 자동 매칭 |
| 진료 절차 이해 부족 및 의사소통 문제 | 동행자가 접수~수납까지 전과정 보조 |
| 병원 운영시간·휴무일 정보 부족 | 실시간 병원/약국 운영정보 API 연동 |
| 보호자 불안감 (위치/진료 확인 불가) | 실시간 위치 추적 및 진료 리포트 제공 |
| 사회적 돌봄 인력 부족 | 시간제 동행자 일자리 창출로 해결 |

> “메디고는 단순한 동행 서비스가 아니라, **의료 접근의 격차를 해소하는 연결 플랫폼**입니다.”

---

## 🧩 서비스 구조 및 운영 방식

- **사용자(보호자)**  
  병원 진료 일정과 위치를 등록하고, 근처에서 활동 중인 동행자를 요청합니다.  
  이후 앱을 통해 진료 진행상황·이동경로를 실시간으로 확인합니다.

- **동행자(매칭 파트너)**  
  사전에 등록·신원 인증·교육 과정을 거친 인원만 서비스 참여가 가능하며,  
  매칭 후 접수~진료~수납~귀가 전 과정을 환자와 함께 수행합니다.

- **앱 운영사(메디고팀)**  
  동행자 인증 및 평가 관리, 보험/안전 정책 마련, 수수료 정산,  
  실시간 서비스 모니터링을 담당합니다.

## ⚙️ 주요 기능

### 🧑‍🤝‍🧑 1. 보호자 1:1 매칭
- 사용자가 동행 요청 시, **가까운 인증된 동행자** 자동 추천  
- 성별·활동 시간·리뷰·인증 상태 기반 필터링  
- 프로필/평점/후기 확인 후 매칭 확정

### 🏥 2. 병원 및 약국 정보 조회
- 공공데이터 API를 통해 **병원 진료 시간, 약국 영업 상태 실시간 제공**  
- 휴무일, 위치, 진료과, 야간 운영 여부 표시  
- “현재 문 연 병원”만 필터링 가능

### 📋 3. 진료 리포트 및 결과 공유
- 동행자가 진료 및 처방 내용을 **요약 리포트** 형태로 보호자에게 전송  
- 진단명, 처방 약품, 의사 소견, 다음 예약일 자동 기록  
- 사진 업로드 및 음성 리포트 기능 지원

### 📍 4. 실시간 위치 공유
- GPS 기반 실시간 위치 공유 및 이동 동선 표시  
- (출발 → 병원 → 약국 → 귀가) 상태 변경 실시간 반영  
- 병원 도착 시 푸시 알림 자동 전송

### 💳 5. 수납 대행 및 정산 시스템
- 동행자가 병원 수납을 대행하고 앱 내 결제 시스템으로 처리  
- 보호자 결제금액 + 서비스 수수료 자동 계산  
- 운영사 수익 일부는 동행자 보상으로 분배

### 🛡️ 6. 안전성 관리
- 모든 동행자는 **보험·신원·범죄이력 조회**를 통과해야 활동 가능  
- 이용자 신고 및 평가 시스템으로 신뢰도 확보  
- 서비스 이용 시 실시간 고객센터 연결 가능

## 설치 및 실행

### 1. 프로젝트 클론

```bash
git clone [repository-url]
cd Medigo
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 추가 의존성 설치

이 프로젝트는 다음 추가 패키지들을 사용합니다:

```bash
# 이미지 선택 기능을 위한 패키지
npm install expo-image-picker

# 또는 expo CLI 사용
expo install expo-image-picker
```

### 4. 앱 실행

```bash
# 개발 서버 시작
npm start

# iOS 시뮬레이터에서 실행
npm run ios

# Android 에뮬레이터에서 실행
npm run android

# 웹 브라우저에서 실행
npm run web
```
